<p class="lede">Every Atto document is a single plain-text file. Writing is handled by <DefineTerm term="Markdown" partOfSpeech="markup language">A plain-text writing format that uses simple symbols (like <code>**bold**</code> and <code># Heading</code>) to mark structure and emphasis while staying readable as raw text. Created in 2004; widely used for documentation, blogs, and notes.</DefineTerm>, math is handled by <DefineTerm term="LaTeX" partOfSpeech="typesetting system" pronunciation="lā-tek">A typesetting system designed for mathematical and scientific documents. Authors write formulas as code (e.g. <code>\frac{`{a}{b}`}</code> for a fraction) and the system renders them as properly typeset notation. The standard for academic publishing since the 1980s.</DefineTerm>, and structure (signatures, credit checks, parties) is handled by Atto directives. The three live together in one file, parsed once, producing one document.</p>

Every Atto Document (AD) is a `.atto` file: human-readable,
git-diffable, and stable under round-trip serialization. Equal ASTs
produce equal byte streams.
