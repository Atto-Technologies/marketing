<p class="lede">Atto Documents combine three layers into a single plain-text format, the Atto Format (AF). Markdown carries prose, LaTeX carries notation, and Atto directives carry structure. The parser walks them in order and produces a single deterministic Abstract Syntax Tree (AST).</p>

Every Atto Document (AD) is a `.atto` file: human-readable,
git-diffable, and stable under round-trip serialization. Equal ASTs
produce equal byte streams.
