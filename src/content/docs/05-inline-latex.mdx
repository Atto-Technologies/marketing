Wrap an expression in single dollar signs: `$…$`. Renders in-line
with surrounding prose at body text size. Dollar signs inside the
expression must be escaped as `\$`.

### Variables and operators

<AttoDemo layout="split" filename="math-simple.atto">
  <article class="atto-doc" slot="rendered">
    <p>The late fee is <span class="atto-math">L = B × 0.015</span> per day. Present value equals <span class="atto-math">PV = FV / (1 + r)<sup>n</sup></span>.</p>
  </article>

  <pre class="code-block" slot="source" set:html={`<code>The late fee is <span class="tok-math">$L = B \\times 0.015$</span> per day.
Present value equals <span class="tok-math">$PV = FV / (1 + r)^n$</span>.</code>`} />
</AttoDemo>
